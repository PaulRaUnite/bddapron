\section{About}

This library is actually composed of two sub-libraries:
\begin{itemize}
\item Bdd: higher-level interface to \cudd (compared to
  \mlcuddidl):
  \begin{itemize}
  \item Manipulation of formulas and expressions of various finite types:
    Boolean, bounded integers, enumerated types;
  \item BDDs as an abstract domain for such datatypes.
  \end{itemize}
\item Bddapron:
  \begin{itemize}
  \item Formulas and expressions mixing finite types and
    numerical types, represented as BDDs/MTBDDs (\textit{e.g.,}
    \texttt{if x>=y-3 and b then x-y else x-2*y*y}), with
  \item Abstract domains combining finite-type and numerical
    variables, by combining BDDs/MTBDDs (for finite-type variables) and \apron domains (for numerical variables) 
  \end{itemize}
\end{itemize}

Both sublibraries require the \camllib utility library, the
\cudd library and \mlcuddidl binding. The Bddapron sublibrary
requires in addition the \apron library and its dependencies
(\gmp,\mpfr,\mlgmpidl).
